This folder contains 3 main codes
1. prepare_and_predict.m is used to prepare data by k-fold (10 folds) and then predict Ihat.
2. cal_performance.m is used to calculate the performance indices of Ihat.
3. plotting codes
3.1 plot_nrmse.m is used to plot bar chart of NRMSE (normalized by mean of I).
3.2 perf_plot_kfold.m is used to plot RMSE, RMSE (specific hour), and MBE (specific hour).
